``komeji-vim``
===

A (neo)vim colorscheme. Made with lush.

<img alt="Screenshot 1" src="https://github.com/isopropyletherperoxide/komeji/blob/main/screenshots/screenshot_1?raw=true" width=256>

### Installation
Use a ``vim`` plugin manager like ``plug``:

```
Plug 'isopropyletherperoxide/komeji-vim'
```

Set colorscheme with

```
colorscheme komeji
```

in the Vim commandline
